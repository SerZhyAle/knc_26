## HUD (верхняя строка)

```
КРОВавица и ЧУДОвище.  1 -ой уровень,  0 -Очей  04:16:30  37 -СТЕН  2 -ЛЕН.
```
- **Уровень** — номер текущего уровня
- **Очей** — текущие очки
- **Время** — реальные часы:минуты:секунды
- **СТЕН** — количество стен на поле
- **ЛЕН** — количество Лен (теней) на уровне

## Визуальный стиль

- Фон: яркий бирюзовый (#00AAAA)
- Стены: тёмно-красные **крестообразные** фигуры (не прямоугольники)
- Выход: **зелёный овал**
- Чудовище (Сергей): маленький пиксельный гуманоид, сиреневый
- Кровавица (Ольга): круглый розово-белый спрайт с лицом
- Лены (тени): похожий гуманоид, фиолетово-коричневый
