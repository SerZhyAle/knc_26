KnC Game History

1999 · DOS

The original game was written by Serge Zhigunenko for DOS. Its visual identity is the bright cyan field, red cross-shaped walls, green exit, and pixel characters.

2003-2005 · Windows

Later Visual Basic versions experimented with multilingual resources, settings, and skin support.

2026 · Web/PWA

The current remake targets TypeScript, Canvas, local saves, offline PWA use, and GitHub Pages distribution.
