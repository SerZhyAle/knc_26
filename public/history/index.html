KnC Game History

From a 1999 DOS survival puzzle by Serge Zhigunenko to the current 2026 Web/PWA remake with TypeScript, Canvas, local saves, Original 1999 graphics, and GitHub Pages deployment.

Version Timeline

13 February 1999 · DOS original

The first version was created by Serge Zhigunenko under the SZA author marker. It used the historical spelling Krovavitsa, arrow-key movement, Q/Esc quit, M sound toggle, N restart, and H to cancel a CPU-speed-dependent auto-move bug.

1999-2001 · Scratch archive

The archive contains helper experiments and configuration material, including a 15x15 field-size config and unrelated UI/media snippets. These files are preserved as context, not as the primary game implementation.

2003 · Visual Basic 6 rewrite

The KNC3 branch moved the game into a Windows GUI. The VB project describes itself as Krovavica & Chudovishe v3 and added ENG.LNG/RUS.LNG, menus, registry-backed settings, and an author string dated 2003.

2003-2005 · Expanded Windows version

The later VB6 implementation became the most complete Windows-era branch. Its form code contains a skin switch, color mode, photo-head GIF sprites, brick walls, a Ukrainian-emblem exit, and five rotating Shadow heads. The small VB.NET branch is preserved as an abandoned attempt.

2026 · Web/PWA MVP

The current version is a browser-first remake. It runs on GitHub Pages, works as a PWA, saves locally in the browser, renders Canvas graphics, and supports Original 1999, Windows 2003 skin, and Modern Color modes.

Rules Summary

GoalMove the Monster to the Exit.
Turn cycleOne player action triggers Monster, then Kryvavitsa, then all Shadows in shuffled order.
MovementOnly four orthogonal directions exist. Diagonal movement and diagonal kills do not exist.
WallsThe Monster pushes exactly one Wall. A Wall can move into an empty cell, crush a Shadow, or disappear into the Exit. Wall chains are blocked.
DefeatMoving onto an enemy loses immediately. Orthogonal adjacency after each enemy action also loses.
VictoryReaching the Exit immediately generates a new board and adds one Shadow to the next level.
ScoreExit +1000, Wall push +10, crushed Shadow +50, each Monster turn -10, restart -500, score cannot go below zero.

Visual Modes

Original 1999Primary MVP mode: extracted DOS-style pixel sprites, bright cyan field, red cross-shaped Walls, green oval Exit.
Windows 2003 skinIntermediate mode promoted from OLD/KnC_NEW/*.GIF: photo-head characters, brick Wall, white tiled field, Ukrainian-emblem Exit.
Modern ColorFallback/debug mode: simple Canvas shapes and colors, useful when image assets fail to load.

Current Implementation

Current MVP version: 26.05.31.2340. The game is implemented with TypeScript, Vite, HTML5 Canvas, Vitest, ESLint, a static service worker, and GitHub Actions deployment to GitHub Pages.

LayerResponsibility

Хронология версий

13 февраля 1999: DOS-оригинал

Архив: OLD/K&C/. Версия 1 использовала историческое написание «Кровавица», авторскую метку SZA, движение стрелками, Q/Esc для выхода, M для звука, N для перезапуска и H для отмены CPU-зависимого авто-движения врагов.

1999-2001: рабочий архив экспериментов

Архив: OLD/prg/. Есть конфиг размера поля и вспомогательные пробы. Это исторический контекст, а не основная игровая реализация.

2003: VB6-переписывание KnC3

Архив: OLD/KNC3/. Visual Basic-проект описывает себя как Krovavica & Chudovishe v3. Появились Windows-формы, файлы ENG.LNG/RUS.LNG, настройки в реестре и авторская строка с датой 2003.

2003-2005: расширенная VB6-версия со скинами

Архив: OLD/KnC_NEW/. Основная форма содержит переключатель skin, цветной режим, режим изображений, кирпичные стены, выход с украинским гербом, голову игрока, голову преследователя и пять чередующихся голов Теней.

2026: Web/PWA-ремейк

Версия 26.05.31.2340. Текущая сборка использует TypeScript, Vite, Canvas, сохранения в localStorage, игровые строки EN/RU/UK, PWA-файлы, GitHub Actions и публикацию на GitHub Pages.

Основные правила

ЦельДовести Чудовище до Выхода.
Цикл ходаОдно действие игрока запускает ход Чудовища, затем Крывавицы, затем всех Теней в перемешанном порядке.
ДвижениеЕсть только четыре ортогональных направления. Диагонального движения и диагонального убийства нет.
СтеныЧудовище толкает ровно одну Стену. Стена может уйти в пустую клетку, раздавить Тень или исчезнуть в Выходе. Цепочки Стен блокируют ход.
ПоражениеХод на врага сразу проигрывает. Соседство по вертикали или горизонтали после каждого хода врага тоже проигрывает.
ПобедаДостижение Выхода сразу создаёт новое поле и добавляет одну Тень к следующему уровню.
СчётВыход +1000, толчок Стены +10, раздавленная Тень +50, каждый ход Чудовища -10, перезапуск -500, счёт не падает ниже нуля.
